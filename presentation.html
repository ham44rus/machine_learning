顧客情報検索・推定AIシステムご提案資料

ビジネスを加速する次世代顧客分析ソリューション

1. システム概要

本提案は、LINEスキマニから取得した「名前・年齢・性別」などの顧客データをもとに、Web検索を活用して、該当人物がどのような人物かを自動で情報収集・整形するシステムの構築に関するものです。

2. システム構成図

LINEスキマニ
(CSV形式)
→
Pythonバックエンド
(FastAPI)
→
外部検索API
(Brave Search API)
→
大規模言語モデル
(OpenAI / Claude)
→
出力(CSV)＋可視化
(スプレッドシート/QuickSight)

3. 機能一覧

機能 内容
📥 顧客情報取得 LINEスキマニからCSVで取得
🔍 検索実行 Brave Search APIを利用して人物に関連する情報を収集
🤖 AI要約 取得した検索結果をLLMで要約・分類（※後の導入想定）
📤 結果出力 CSV形式で出力：ワーカーの名前、年齢、過去の就業履歴、就業回数、検索結果の内容を記載
📊 データ可視化 Googleスプレッドシート / AWS QuickSightで可視化（レポート用）

4. 活用用途

👥

潜在顧客の興味関心・職業などの自動推定

人物名から業界・経歴などを自動収集し顧客像を明確化

⚡

ヒアリングにおける事前情報収集の効率化

商談前の基礎情報収集を自動化し、準備時間を大幅短縮

📈

コンバージョン率推定のためのデータベース構築

顧客属性と成約率の相関を分析し最適化

5. 使用予定技術

項目 ツール・技術
検索API Brave Search API（初期段階）
※効果検証後にGoogle API / Tavily APIへ拡張可能
AI要約 OpenAI GPT-4 / Claude 3（必要に応じて段階的に導入）
バックエンド Python（FastAPI）
データ保存 AWS等のクラウドシステム（S3など）へ保存
ローカルCSVも選択可能
データ可視化 Googleスプレッドシート / AWS QuickSightを活用
出力形式 CSVファイル（ワーカー情報＋検索内容含む）
